# certipy

## certipy - find vulnerable templates
```
# certipy - find vulnerable templates - tips
- certipy -vulnerable options relies on your user's rights to list certificates availables to him that are vulnerable. As such, it does not show other certificates available for other uses that you can abuse.
  - use certipy w/ vulnerable option
  - list every certs
  - find abusable ones
  - check the Groups/user in enrollement rights
  - target those users
```
